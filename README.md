# About this repository

This repository is a static research compendium associated with the preprint "Evaluation of a structure-based method for ab initio gene detection using deep learning" by Hummel and Estrada (DOI: [10.64898/2025.12.19.694709](https://doi.org/10.64898/2025.12.19.694709 )). It is currently **unsupported**; while the code is fully functional for reproducing the reported results, no further features or updates are planned.

That [preprint](https://doi.org/10.64898/2025.12.19.694709 ), and this supporting repository, encapsulate the final version of the work conducted during my capstone research project in the Brandeis University Bioinformatics MS program. The models trained and evaluated during this project were later put into a command-line tool, called [Helix-EXon-finder](https://github.com/GreenMtn-bioinfo/Helix-EXon-finder). Check out that repository for a more organized and accessible way to explore the pipeline's current capabilities and underlying code base.
