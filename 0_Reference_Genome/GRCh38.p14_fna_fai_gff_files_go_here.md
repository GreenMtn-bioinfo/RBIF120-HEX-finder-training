# Context on this directory

+ This is where the necessary reference genome files (sequences, index, and annotation) should be stored
    + They are not included in this repository or Zenodo, but are publicly available at the NCBI website

+ You can also automatically fetch/prepare copies of these files if you run the included [fetch_reference.sh](/0_Reference_Genome/fetch_reference.sh) script from within this directory with the provided conda environment built and activated (see [README.md](/README.md))
